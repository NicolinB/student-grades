# student_grades

CLI application for managing student grades written in C.

## Compile & Run

```bash
gcc student_grades.c -o student_grades
./student_grades
```

## Features

- Add / remove students
- Enter multiple grades per student
- Sort students by average (descending)
- Search by partial name
- Class statistics (best, worst, class average, pass rate)
- Auto-save and load from `grades.dat`
